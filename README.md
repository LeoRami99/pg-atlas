## Quick start 🏄

Prerequisites: [Node (v20 LTS)](https://nodejs.org/en/download/) and [Git](https://git-scm.com/downloads)

> Clone the repository:

```
git clone https://github.com/LeoRami99/pg-atlas.git
```

> Install dependencies:

```
bun
```

> Run the project:

```
bun run dev
```
